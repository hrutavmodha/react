# JSX DOM Kit

- **JSX DOM Kit** is a to-be NPM library through which the UIs can be made efficiently

- By using the kit, you can make the Declarative UI for your websites through JSX, just like **React** or **Vue**

- This library is build using TypeScript for avoiding any run-time type errors

- The features of the kit are as follows:

## Features

1. JSX-based Declarative UI-building

2. Component nesting via JSX is possible

3. Client-side routing support

4. Base component functions being close to DOM, does not take much time to load the website initially

5. Minimal dependency
    - Just uses Vite and TypeScript for development purpose
